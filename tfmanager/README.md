
tfmanager
======

A text file Manger

The code is Python 3 compatible.

Installation
============

Fast install:

```
pip install tfmanager

```

For a manual install get this package:

```
wget https://github.com/hsdoaa/tfmanager/archive/master.zip
unzip master.zip
rm master.zip
cd tfmanager-master
```

Install the package:
```
python setup.py install    
````
